Mac Key Hints - [open/close]
  • click - select classes/attr./oper.
  • del(fn+back) - delete selected item
  • ALT-click - delete association
  • cmd + left-click - change association direction
  • right-click - change type of association
  • right-click + cmd - change to dependency
  • double-click on association- add/edit label
  • SHIFT+click on association- add/edit multiplicity
